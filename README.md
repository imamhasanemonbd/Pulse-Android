# Pulse - High-End Native Music Streaming App

Pulse is a modern, high-end native music streaming application for Android. Built with a focus on aesthetics, performance, and resilience, it delivers a premium listening experience using Material 3 Expressive UI and the latest Android media technologies.

## ✨ Features

- **Material 3 Expressive UI**: A beautiful, fluid interface following the latest Material Design standards with an 8/16dp grid.
- **Resilient Streaming**: Powered by a robust InnerTube-based backend for reliable music playback.
- **Media3 Playback**: Advanced background playback support using Android's Media3 library (ExoPlayer).
- **Smooth Navigation**: Fast and intuitive navigation with optimized network and data layers.
- **Personalization**: History tracking and library management using Room persistence.
- **Visualizations**: Dynamic audio visualizers that react to your music in real-time.

## 🛠️ Tech Stack

- **UI**: Jetpack Compose, Material 3 Expressive
- **Media**: Android Media3 (ExoPlayer), MediaSession
- **Networking**: Retrofit, OkHttp, InnerTube Integration
- **Database**: Room Persistence Library
- **Architecture**: MVVM with a focus on Clean Architecture principles
- **Image Loading**: Coil

## 🚀 Getting Started

To get started with Pulse, clone this repository and open it in Android Studio.

```bash
git clone https://github.com/imamhasanemonbd/Pulse-Android.git
```

### Prerequisites

- Android Studio Koala (2024.1.1) or higher
- Android SDK 24+ (Min SDK)
- Java 17
